Python PyGame Linear Algebra Math 2D-Projection

Plane Projection

This is a simple program that uses linear algebra concepts to project a 3d shape onto a 2d plane.
In this program, the shape (which is a cube) can be rotated around, and it can be adjusted. Moreover, since it uses the idea of projection from a perspective, you can adjust the distance of the projection.

Python Random Path Visualization

Random Walker Visualization

This is a little visualization of a cool random walk that uses a circle technique that I found intriguing. The technique gives the walk a more natural or fluent randomness in its steering behavior. Essentially, this is a very small and simple implementation that is somewhat similar to Craig Reynolds's steering behavior - wander.
